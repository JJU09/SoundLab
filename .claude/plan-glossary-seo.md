# 용어집 "X란?" SEO 플랜

> 목적: 저경쟁 한국어 음악/사운드 용어별 검색 진입점 신설(+40 URL). 토픽·도구·샌드박스와 상호링크.
> 차별점: 대부분 "X란?" 페이지는 텍스트뿐 — 우리는 **라이브 위젯 임베드** → 듣고/만지며 이해 → 체류↑·바운스↓ → 고유성·순위 유리.

## 라우팅·콘텐츠 모델
- 경로: `/glossary/[slug]/` (영문 slug) + `/glossary/` 인덱스
- 신규 콘텐츠 컬렉션 `glossary` (토픽과 분리). 스키마:
  - `term`(en) · `kr` · `hub` · `definition`(1~2문장, 답변 먼저) · 본문 MDX(짧게)
  - `widget?`(WIDGETS enum 재사용 — TopicWidget 임베드) · `widgetConfig?`
  - `relatedTopic` · `relatedSandbox` · `aliases[]`(검색 변형어)
- **얇은/중복 금지**: 토픽 복붙 ❌. 용어집 = 간결 정의(100~250자) + 라이브 위젯 + "더 알아보기"로 토픽 링크. 깊이는 토픽, 진입은 용어집.

## 페이지 SEO 구조 (per 용어)
- title: `에일리어싱이란? — 디지털 오디오 용어 · SoundLab` (검색어 그대로)
- H1: `에일리어싱이란?` → 바로 아래 1~2문장 정의(피처드 스니펫 노림)
- 짧은 본문: 왜/예시
- 라이브 위젯 임베드(있으면)
- JSON-LD `DefinedTerm`(+ 가능시 FAQPage)
- 내부링크: 토픽 · 샌드박스 · 관련 용어 2~3개

## 용어 선정 (검색 수요 우선순위)
- **1순위(고검색·입문)**: 데시벨/dB · BPM · 옥타브 · 헤르츠 · 화이트노이즈/핑크노이즈 · A440(튜닝) · 사인파/톱니파/구형파 · 모노/스테레오
- **2순위(토픽 직결)**: 에일리어싱 · 샘플레이트 · 비트뎁스 · ADSR · LFO · 컴프레서 · EQ · 리버브 · 컷오프/레조넌스 · 팬 · 라우드니스/LUFS · 클리핑/헤드룸 · 배음(하모닉스) · 맥놀이
- **3순위(보강)**: 나이퀴스트 · 센트 · 게인 · 펜타토닉 · 콤필터 · 위상 등
- 총 30~45개 목표.

## 구현 단계
- A. 인프라: `glossary` 컬렉션 + `GlossaryLayout.astro`(정의-우선·JSON-LD·위젯·크로스링크) + `/glossary/` 인덱스(허브별 분류·내부링크 허브) + 샘플 1개. 빌드·검증.
- B. 1~2순위 ~20개 작성(검색 수요 높은 것부터).
- C. 나머지 + 토픽↔용어 상호링크(토픽 본문 인라인 링크 → 내부링크 웹 강화).

## 네비/노출
- 푸터 또는 "지식 허브" 드롭다운에 용어집 추가
- sitemap 자동 포함(45 → ~85 URL)

## 측정
- Search Console에서 "X란/뜻" 쿼리 노출·순위 추적 → 먹히는 용어 보강

## 비고
- 핵심 효과: 저비용 검색 진입점 40개 + 라이브 위젯 차별화.
- 소셜(스파이크)과 달리 검색(상시) 채널을 키우는 작업 — 트래픽 지속성의 핵심.
